# 角色名快捷输入（QuikInput）

一个绑定 SillyTavern 角色卡的快捷输入扩展。切换角色卡时，输入框上方的按钮会自动切换。

## 功能

- 每张角色卡保存独立按钮列表
- 全局扩展开关
- 支持任意数量的按钮
- 点击后可在光标处插入、末尾追加或替换全部输入
- 配置写入角色卡 `data.extensions.quikinput`，可随角色卡导出

## 安装

1. 将本目录提交并推送到 GitHub 仓库。
2. 打开 SillyTavern 的“扩展”。
3. 选择“安装扩展”，粘贴 GitHub 仓库链接。
4. 安装完成后刷新页面。

## 使用

1. 打开“扩展”面板中的 **Character Quick Input**。
2. 开启“启用扩展”。
3. 选择要编辑的角色卡并添加按钮。
5. 切换到对应角色聊天，按钮会显示在酒馆原生的 Quick Reply（QR）栏内。

当前版本仅处理单角色聊天，群聊中会隐藏按钮。
